Test Points Manifest

This manifest is auto-generated from the centralized test IDs in lib/testIds.ts

Usage

Load this manifest in your test scripts to get consistent selectors:

// Load test manifest
const response = await fetch('/test-manifest.html');
const html = await response.text();
const parser = new DOMParser();
const doc = parser.parseFromString(html, 'text/html');
const manifest = JSON.parse(doc.getElementById('test-manifest').textContent);

// Use selectors
const logoSelector = manifest.testPoints.navigation.elements.logo.selector;
